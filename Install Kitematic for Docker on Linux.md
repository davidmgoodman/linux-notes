<h2><a href="#ubuntu" aria-hidden="true" class="anchor" id="user-content-ubuntu"><svg aria-hidden="true" class="octicon octicon-link" height="16" version="1.1" viewBox="0 0 16 16" width="16"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Ubuntu</h2>
<p>Download the Ubuntu package (Kitematic-X.X.X-Ubuntu.zip) from the <a href="https://github.com/docker/kitematic/releases/">release page</a>.</p>
<p>Assuming that you are already in the right directory extract the archive and
install kitematic issuing the following command:</p>
<pre><code>$ sudo dpkg -i ./Kitematic_X.X.X_amd64.deb

<pre><code>$ sudo groupadd docker

$ sudo gpasswd -a ${USER} docker

$ sudo service docker restart
</code></pre>
<p>To activate the changes to docker group either do a <code>$ newgrp docker</code>, in
this case you will be able to properly run kitematic only from terminal
otherwise you will receive the following message:</p>
<p><a href="/max-devjs/kitematic.wiki-my/blob/master/img/04-kitematic.png" target="_blank"><img alt="kitematic setup initialization" width="600" src="/max-devjs/kitematic.wiki-my/raw/master/img/04-kitematic.png" style="max-width:100%;"></a></p>
<p>or log out/in (in this case you will then be able to run kitematic from
terminal, graphical menu, etc).</p>
